## Installation

1. Make sure [`vue-i18n`] is installed in your root directory
2. Install this extension
3. Configure your `locales` path. There are three ways to do that:
   - Open **Command Palette** (`Ctrl-Shift-P` or `‚åò‚áßP`), type `VueI18n Ally: Auto detect locales path` then press enter.
   - Open **Command Palette** (`Ctrl-Shift-P` or `‚åò‚áßP`), type `VueI18n Ally: Manual configure locales path` then press enter and follow the guide.
   - Modify the `settings.json` file of your VSCode, adding `vue-i18n-ally.localesPath` manually.
